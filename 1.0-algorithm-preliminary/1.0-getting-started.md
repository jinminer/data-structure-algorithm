# 概念

## 定义

数据结构与算法的定义(广义): 

* 数据结构: 一组数据的存储结构
  * 数据在内存中是以何种方式存储的
  * 数据结构是静态的，它只是组织数据的一种方式
* 算法: 操作数据的一组方法

## 关系

数据结构是为算法服务的，算法要作用在特定的数据结构之上。

# 要点

## 复杂度分析

* 数据结构和算法解决的是如何更省、更快地存储和处理数据的问题

* 复杂度分析是用来度量数据结构和算法如何更省、更快地存储和处理数据的方法，需要考虑到效率和资源消耗的问题。
